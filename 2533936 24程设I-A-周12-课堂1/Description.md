# 子字符串

编写一个函数，用于检查字符串 s1 是否是字符串 s2 的子字符串。如果匹配，函数返回 s2 中 s1 首次出现的索引；否则，返回 -1。

函数的定义如下：

```C
int indexOf(const char *s1, const char *s2)
```

# 输入

输入是两个 C 风格字符串 s1和s2。

# 输出

输出是s1作为s2的子字符串首次出现的索引。如果s1不是s2的子字符串，则返回-1。

# 示例输入1

```
Abc
eAbc
```

# 示例输出1

```
1
```

# 示例输入2

```
aBd
a
```

# 示例输出2

```
-1
```
