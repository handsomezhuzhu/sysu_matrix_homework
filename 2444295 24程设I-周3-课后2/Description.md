# 24程设I-周3-课后2

## 题目

现在想要加权求出一个总和，加权的公式如下$w_1 * a +  w_2 * b + w_3 * c = T$, 其中$w_1,w_2,w_3$是浮点数，表示权重, a,b,c都是整数，T为计算后得到的总和，也是浮点数。现在，请编写一个程序来求出这个总和T。

## 问题输入

输入共三行，每一行分别有浮点数$w_1$和整数a。

## 问题输出

输出总价，保留两位小数。

## 样例输入

```
0.50 3
0.20 2
0.70 1
```

## 样例输出

~~~
2.60
~~~

## 提示

要选择对变量类型和格式化控制符。本题请使用单精度浮点数类型变量。
