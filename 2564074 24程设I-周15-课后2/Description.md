# 24程设I-周15-课后2

# 问题描述

要求设计一个C语言函数，实现删除链表中所有重复元素的节点，使每个元素只出现一次，返回无重复元素的链表（不改变原始节点相对顺序，且重复元素按从左至右顺序保留第一个）

# 输入描述

1. 第一行输入链表节点个数`n`，范围为[2, 50]
2. 第二行输入所有链表节点数据，范围为[1, 100]

# 输出描述

1. 第一行输出原始链表
2. 第二行输出无重复元素的链表

# 输入示例

```
5
10 20 10 30 30
```

# 输出示例

```
10->20->10->30->30->NULL
10->20->30->NULL
```
