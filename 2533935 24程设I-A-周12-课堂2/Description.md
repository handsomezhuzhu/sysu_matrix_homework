# 24程设I-A-周12-课堂2

# 题目描述

根据主函数和头文件提示，编写子函数`char *revStr(char *s);`，其功能为：将字符串 $s$ 逆序。

# 输入格式

一个长度不超过100的字符串 $s$，可能包括大写字母、小写字母和阿拉伯数字。

# 输出格式

字符串 $s$ 的逆序。

# 输入样例1

```
apple
```

# 输出样例1

```
elppa
```

# 输入样例2

```
programming
```

# 输出样例2

```
gnimmargrop
```

# 提示

若需要返回局部变量地址，需将其定义为static类型；
