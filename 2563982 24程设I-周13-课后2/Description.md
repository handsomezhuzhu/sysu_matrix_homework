# 题目描述

编写函数`void myshift(char *s)`,在不打乱s原本相对位置情况下，将偶数位上的字符全部挪到奇数位字符的前面。注意，下标位数从0开始计算。

# 输入格式

*输入一个字符串 $s$*

**保证输入字符串 $s$ 的长度大于等于1小于等于100**

# 输出格式

输出修改后的字符串 $s$。

# 输入样例1

```012345```

# 输出样例1

```024135```

# 输入样例2

```01234```

# 输出样例2

```02413```

# 输入样例3

```0```

# 输出样例3

```0```
